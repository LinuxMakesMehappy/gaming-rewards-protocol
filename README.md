## 🚀 **Quick Start**

### **Prerequisites**
```bash
# Install Rust (for zero-CVE liquidity engine)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Install Node.js (for TypeScript components)
node --version  # >= 18.0.0

# Install Solana CLI
sh -c "$(curl -sSfL https://release.solana.com/v1.17.0/install)"
```

### **Installation**
```bash
# Clone the repository
git clone https://github.com/your-username/gaming-rewards-protocol.git
cd gaming-rewards-protocol

# Install dependencies
npm install

# Build zero-CVE liquidity engine
cd zero-cve-liquidity-engine
cargo build --release
cd ..

# Build TypeScript components
npm run build
```

### **Configuration**
```bash
# Copy environment template
cp env.example .env

# Configure your environment variables
nano .env
```

### **Running the Protocol**
```bash
# Start the development server
npm run dev

# Run security audit
npm run security-audit

# Run tests
npm test
```
